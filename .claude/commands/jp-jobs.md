일본 기업 사이트에서 채용공고를 찾아 한국어로 요약합니다.

입력: $ARGUMENT = 일본 기업 사이트 URL (예: https://example.co.jp)

## 실행 순서

**1단계 — 채용 페이지 탐색**
WebFetch로 다음 경로를 순서대로 시도해 채용 정보가 있는 페이지를 찾으세요:
- `$ARGUMENT` (메인 페이지 — 채용 링크 확인)
- `$ARGUMENT/careers`
- `$ARGUMENT/recruit`
- `$ARGUMENT/jobs`
- `$ARGUMENT/採用情報`
- `$ARGUMENT/採用`
- `$ARGUMENT/hr`

메인 페이지에서 채용 관련 링크(採用, キャリア, 求人, Recruit, Careers 등)를 발견하면 해당 링크를 우선 따라가세요.

**2단계 — 공고 수집**
채용 페이지에서 개별 공고 링크를 추출하고, 각 공고 페이지를 WebFetch로 가져오세요.
공고가 많을 경우 최대 10개까지만 처리하세요.

**3단계 — 한국어 요약 출력**
아래 형식으로 각 공고를 요약하세요:
